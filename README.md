# Sistema de Controle de Vendas
---
# Objetivo do sistema
- Criar um sistema para controlar vendas, clientes, produtos, estoque e relatórios financeiros. O sistema usará Python para a programação, MySQL para armazenar os dados e Excel para gerar relatórios profissionais.

---
# Tecnologias usadas
Python: linguagem principal do sistema;

MySQL: banco de dados;

Excel: relatórios gerenciais;

Pandas: tratamento dos dados;

OpenPyXL: criação de planilhas Excel;

Tkinter ou Streamlit: interface do sistema;

MySQL Connector: conexão entre Python e MySQL;
